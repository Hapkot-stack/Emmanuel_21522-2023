Responsive Personal Portfolio Website
A responsive personal portfolio website built with HTML, CSS, and JavaScript as required for the Web Technology assignment.

Technical Stack
• HTML for structure
• CSS for styling
• JavaScript for interactivity
Portfolio Content
• Photo and bio section
• Education background
• Experience (internships, jobs, volunteer work)
• Projects (at least two with descriptions)
• Skills (with progress bars)
• Certificates and awards
• Contact information (phone, email, LinkedIn, GitHub)
• Contact form (name, email, message)


Design and Layout
• Consistent color scheme and typography
• Navigation bar for sections
• CSS Flexbox 
• Fully responsive (media queries)
• Semantic HTML elements

Bonus Features
• Hover effects and transitions
• FontAwesome icons
• Smooth scrolling
• Animated skill bars
• Mobile hamburger menu

Deployment
GitHub Repository
• Create a repository named: YourName_RegNumber
• Upload all project files
• Ensure index.html is in the root folder
GitHub Pages
• Open repository settings
• Go to "Pages"
• Select "Deploy from branch"
• Choose the main branch
• Save and wait for live deployment
